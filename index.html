Plataforma web vs escritorio

Ventajas
y desventajas

APLICACION WEB
Ventajas:
+ Se puede usar desde cualquier lugar.
+ No requiere hacer actualizaciones en los clientes.
+ No hay problemas de incompatibilidad entre versiones, porque todos trabajan con la misma.
+ Se centralizan los respaldos de información.
Desventajas:
- Requiere conexión a la red.
- Su tiempo de respuesta podría ser más lento que una aplicación de escritorio.

APLICACIÓN DE ESCRITORIO
Ventajas:
+Pueden ser más robustas.
+Tiempo de respuesta más rápido.
Desventajas:
-Requiere instalación en cada cliente.
-Generalmente se hacen para un sistema operativo específico.
-Se requiere actualizar en cada cliente.

Características

instalación y actualización web

En la web no requiere instalación todas las actualizaciones se realizan en el servidor. Fuente Appmaster

Instalación y actualización escritorio

Se debe instalarse en una computadora o dispositivo móvil, y debe actualizarse cada vez que se lanza una nueva versión. Fuente Appmaster

Fiabilidad web

Velocidad de la conexión a Internet y del rendimiento del servidor remoto. Fuente Appmaster

Fiabilidad escritorio

Funciona de forma autónoma, por lo que lo principal es la calidad del código y la estabilidad del hardware sobre el que se ejecuta este código. Fuente Appmaster

Multiplataforma web

Funcionará igualmente bien en cualquier dispositivo, puede ser una computadora estacionaria, computadora portátil, tableta o teléfono inteligente. Fuente Appmaster

Multiplataforma escritorio

Depende del sistema operativo, procesador, tarjeta de video y varios otros parámetros. Fuente Appmaster

seguridad web

Desarrollada con protocolos modernos y herramientas de seguridad puede garantizar completamente la seguridad de los datos. Fuente Appmaster

Seguridad escritorio

Es configurable de manera más flexible, lo que significa que teóricamente todas las vulnerabilidades potenciales se pueden prever durante su desarrollo. Fuente Appmaster
